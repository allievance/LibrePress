# LibrePress

LibrePress is a lightweight, end-to-end publishing system for producing and distributing small-run books without inventory risk.

It replaces traditional publishing overhead with a streamlined, developer-controlled pipeline: manuscript → formatting → design → print-on-demand fulfillment.

## Philosophy

LibrePress is built on three principles:

- **Clarity over complexity** — simple, inspectable systems
- **Automation over overhead** — eliminate manual publishing workflows
- **Ownership over platforms** — retain control of infrastructure, data, and distribution

This is not a marketplace. It is a system.

## Features

- 📄 Manuscript ingestion and normalization
- 🎨 Programmatic layout and cover generation
- 📦 Print-on-demand integration (e.g., Lulu)
- 🔗 Order automation pipelines (API-first)
- 🧩 Modular architecture (Flask-based)
- ⚙️ Designed for self-hosting and extensibility
